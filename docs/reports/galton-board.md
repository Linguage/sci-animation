## 理论依据

高尔顿板把“许多次独立随机选择”的结果变成可见的形状。小球每经过一层钉子，就以概率 `p` 向右、以概率 `1-p` 向左。若共有 `n` 层，最终落入第 `k` 个槽位等价于在 `n` 次试验中向右 `k` 次：

```text
P(X = k) = C(n, k) p^k (1-p)^(n-k)
```

因此底部槽位计数服从二项分布。当 `p = 0.5` 时，分布关于中间槽位对称。随着小球数量增加，实际堆积会围绕理论二项分布波动。

当层数较多时，二项分布可以用正态分布近似：

```text
mu = np
sigma^2 = np(1-p)
```

这就是底部小球堆积逐渐形成钟形包络线的原因。

## 动画表达

画面中部显示三角钉阵，小球从顶部进入，并沿预生成的左右分岔路径下落。底部柱状槽位显示累计小球数量，黄色曲线表示理论二项分布，灰色曲线表示正态近似。

第一版使用离散路径动画，不进行真实碰撞动力学求解。这样可以保持随机过程可复现，也让观众把注意力放在“独立随机选择如何累积成分布形状”上。
