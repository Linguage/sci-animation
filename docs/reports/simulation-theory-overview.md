# 三个模拟动画的理论依据

> 最近更新：2026-05-28  
> 范围：曲柄摇杆机构、单自由度弹簧振子、三层剪切楼地震响应

本文整理当前项目中三个模拟动画的理论基础、建模假设与动画表达方式。它不是代码说明，而是回答“这些画面背后的力学模型是什么”。

## 1. 曲柄摇杆机构

### 动画

![曲柄摇杆机构动画预览](../../media/previews/crank_rocker.gif)

高清视频文件：[`CrankRockerScene.mp4`](../../media/videos/crank_rocker/480p15/CrankRockerScene.mp4)

### 理论模型

曲柄摇杆机构是平面四杆机构的一种。当前模型将四个铰点记为：

- `A`：曲柄固定铰
- `B`：曲柄与连杆连接点
- `C`：连杆与摇杆连接点
- `D`：摇杆固定铰

机架 `AD` 固定，输入为曲柄角 `theta`。在任意时刻，`B` 点由曲柄长度和输入角直接确定：

```text
B = A + r * [cos(theta), sin(theta)]
```

`C` 点同时满足两个长度约束：

```text
|C - B| = coupler
|C - D| = rocker
```

因此 `C` 是以 `B` 为圆心、连杆长为半径的圆，与以 `D` 为圆心、摇杆长为半径的圆的交点。动画中的四杆求解本质上就是圆-圆相交问题。

### 建模假设

- 所有杆件视为刚体。
- 所有连接为理想转动铰。
- 不考虑质量、惯性、摩擦和驱动力矩。
- 输入角按给定速度匀速变化，属于运动学演示，不是动力学仿真。

### 动画表达

动画展示了：

- 曲柄连续转动。
- 连杆带动摇杆往复摆动。
- `C` 点运动轨迹。
- 输出摇杆角 `theta_out` 的实时变化。

这个示例的核心价值是建立“机构约束决定运动”的直觉：一根杆的输入角确定后，其它杆的位置由几何约束解出。

## 2. 单自由度弹簧振子

### 动画

![单自由度弹簧振子动画预览](../../media/previews/spring_oscillator.gif)

高清视频文件：[`SpringOscillatorScene.mp4`](../../media/videos/spring_oscillator/480p15/SpringOscillatorScene.mp4)

### 理论模型

单自由度弹簧振子由一个质量块和一个线性弹簧组成。自由度为质量块相对平衡位置的水平位移 `x(t)`。

无阻尼、无外力自由振动的控制方程为：

```text
m * x_ddot + k * x = 0
```

其中：

- `m`：质量
- `k`：弹簧刚度
- `x`：位移
- `x_ddot`：加速度

固有圆频率为：

```text
omega_n = sqrt(k / m)
```

若初始相位为 `phi`，振幅为 `A`，位移响应可写成：

```text
x(t) = A * cos(omega_n * t + phi)
```

加速度由位移二阶导得到：

```text
x_ddot(t) = -omega_n^2 * x(t)
```

因此加速度与位移始终反相，且幅值放大倍数为 `omega_n^2`。

### 建模假设

- 弹簧为线性弹簧，满足 Hooke 定律。
- 质量块为集中质量。
- 不考虑阻尼、摩擦和外部激励。
- 运动只有一个水平自由度。

### 动画表达

动画同时展示：

- 左侧质量块随 `x(t)` 做简谐运动。
- 弹簧随质量块伸缩。
- 位移箭头显示当前自由度方向和大小。
- 右侧上图绘制 `x(t)` 时程。
- 右侧下图绘制 `x_ddot(t)` 时程。

这个示例的核心价值是把“自由度响应”和“时程曲线”并置起来：左侧让人看到系统如何动，右侧让人看到同一个自由度在时间域中的数学描述。

## 3. 三层楼地震响应

### 动画

![三层剪切楼地震响应动画预览](../../media/previews/three_story_earthquake.gif)

高清视频文件：[`ThreeStoryEarthquakeScene.mp4`](../../media/videos/three_story_earthquake/480p15/ThreeStoryEarthquakeScene.mp4)

### 理论模型

三层楼模型采用剪切楼假设。每层楼板视为集中质量，层间柱或抗侧构件视为水平剪切弹簧。系统有三个水平自由度：

```text
u = [u1, u2, u3]^T
```

其中 `ui` 是第 `i` 层相对地面的水平位移。

地震输入采用地面加速度 `u_g_ddot(t)`。相对位移形式的结构动力方程为：

```text
M * u_ddot + C * u_dot + K * u = -M * 1 * u_g_ddot(t)
```

其中：

- `M`：质量矩阵
- `C`：阻尼矩阵
- `K`：刚度矩阵
- `1`：全 1 影响向量，表示地面运动同时作用于所有楼层质量
- `u_g_ddot(t)`：地面加速度输入

三层剪切楼的刚度矩阵由各层层间刚度组装。若层刚度为 `k1, k2, k3`，则：

```text
K = [
  k1 + k2,   -k2,       0
      -k2, k2 + k3,   -k3
        0,     -k3,    k3
]
```

当前实现使用 Rayleigh 阻尼：

```text
C = alpha * M + beta * K
```

`alpha` 和 `beta` 由目标阻尼比及前两阶频率确定。

时程积分使用 Newmark 平均加速度法。该方法在结构动力学中很常见，适合线性系统的地震响应计算。

### 建模假设

- 楼板为刚性楼板，质量集中在楼层高度处。
- 结构侧向变形由层间剪切变形主导。
- 每层只有一个水平平动自由度。
- 材料和构件保持线性。
- 地震输入为项目内合成加速度波，不代表某条真实地震记录。
- 动画中的楼层侧移经过放大，便于观察。

### 动画表达

动画展示了：

- 左侧三层楼在地震激励下产生侧向响应。
- 地面加速度输入用底部箭头和红色时程曲线表示。
- 右侧主图绘制 `u1(t)`, `u2(t)`, `u3(t)` 三条楼层相对位移时程。
- 结构顶部用 `u3(t)` 标注屋顶位移。

这个示例的核心价值是把“多自由度结构响应”拆成两个可观察对象：结构形态随时间变化，以及每一层自由度的时程响应。

## 4. 三个模型的层级关系

三个动画可以看作从几何运动学到结构动力学的逐步升级：

| 示例 | 模型类型 | 自由度/状态 | 控制关系 | 当前用途 |
|------|----------|-------------|----------|----------|
| 曲柄摇杆机构 | 平面机构运动学 | 输入角 + 铰点坐标 | 几何约束 | 解释机构如何由杆长约束决定运动 |
| 弹簧振子 | 单自由度动力学 | `x(t)` | 常微分方程 | 解释自由度、位移与加速度时程 |
| 三层剪切楼 | 多自由度结构动力学 | `u1(t), u2(t), u3(t)` | 矩阵动力方程 | 解释地震输入下楼层响应 |

它们共同构成一个很适合教学的序列：

1. 先用曲柄摇杆说明“运动可以由约束决定”。
2. 再用弹簧振子说明“一个自由度的动力响应如何随时间变化”。
3. 最后用三层楼说明“多个自由度耦合后如何形成结构响应”。

## 5. 当前实现边界

当前动画以“清晰表达理论结构”为目标，并非工程设计软件。需要注意：

- 曲柄摇杆没有进行力分析和动力学分析。
- 弹簧振子暂未加入阻尼、外力或频响分析。
- 三层楼采用线性剪切楼模型，没有考虑楼板转动、构件非线性、土-结构相互作用或真实地震记录。
- 所有动画中的位移显示都可能为了视觉效果进行放大。

后续若要提高工程可信度，可以加入：

- 曲柄摇杆的速度、加速度和力矩分析。
- 弹簧振子的阻尼自由振动、受迫振动和共振演示。
- 三层楼的模态分析、反应谱对比、真实地震波输入和层间位移角展示。
