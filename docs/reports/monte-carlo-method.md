## 理论依据

蒙特卡罗方法用随机样本近似确定性问题的统计量。本示例在边长为 2 的正方形内均匀采样点，并判断样本是否落在单位圆内：

```text
x^2 + y^2 <= 1
```

正方形面积为 4，单位圆面积为 pi。若随机点在正方形内均匀分布，则落入圆内的概率为：

```text
P(inside) = pi / 4
```

因此，当总样本数为 `N`、圆内样本数为 `N_in` 时，可以用下式估计 pi：

```text
pi_hat = 4 * N_in / N
```

根据大数定律，随着 `N` 增大，样本比例 `N_in / N` 会逐渐靠近真实概率，`pi_hat` 也会围绕真实的 pi 收敛。

## 动画表达

画面左侧显示正方形与内切圆，随机点逐步出现：蓝色表示落在圆内，橙色表示落在圆外。右侧绘制 `pi_hat(N)` 的时程曲线，并用虚线标出真实的 pi。实时读数显示当前样本数、估计值和圆内样本比例。

这个动画的重点不是追求高精度数值计算，而是展示“随机样本比例如何逼近几何面积比例”的思想。后续可以复用同一模式扩展到随机积分、可靠度估计、随机变量分布采样等主题。
