## Phase 1: 曲柄摇杆示例打磨

- [ ] 增加尺寸标注、角度弧线和关键位置说明。
- [ ] 增加参数切换示例：open / crossed 装配支路。

## Phase 2: 机构库扩展

- [x] 增加单自由度弹簧振子演示。
- [x] 增加三层剪切楼地震响应演示。
- [x] 增加蒙特卡罗方法估计 pi 演示。
- [x] 增加高尔顿板二项分布演示。
- [ ] 增加滑块曲柄机构。
- [ ] 增加齿轮传动演示。
- [ ] 增加凸轮从动件演示。
- [ ] 为每个机构补最小几何验证。

## Phase 3: 动画库基础层沉淀

- [ ] 明确项目定位：教学模型 + 仿真结果再现 + 可视化组件，不替代专业仿真软件。
- [ ] 设计统一响应数据结构，例如 `TimeResponse`、`GeometryState`、`ModeShape`。
- [ ] 抽出可复用时程曲线组件，供弹簧振子、剪切楼、后续梁/桥响应共用。
- [ ] 抽出实时数值面板、动态图例、动态追踪点等 Manim 组件。
- [ ] 将 Newmark 等小型求解器从具体结构模型中拆出，形成教学用 `solvers/`。
- [ ] 建立 `replay/` 层，用于读取 CSV/JSON/HDF5 等外部仿真结果并标准化。
- [ ] 为“新增动画”建立模板：模型/数据入口、响应对象、场景、验证命令、GIF 预览、理论报告。

## Phase 4: 仿真再现场景支持

- [ ] 支持从 CSV 导入时程响应并生成标准动画。
- [ ] 支持外部节点位移数据的变形形状重放。
- [ ] 支持真实或外部地震波输入文件。
- [ ] 为复杂项目建立结果目录约定和元数据说明。
- [ ] 增加一个“外部仿真结果重放”的示例场景。
